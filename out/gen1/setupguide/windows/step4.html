  1. User info

    1. - كل ما تلاقي كلمة 'interface' بين قوسين بالشكل ده { interface } في الكلام الجاي هتستبدلها بإسم الinterface بتاعتك و تكتبها بين (") زي كده مثلا: "Wi-Fi"

  2. User info

    2. - هتستبدل primary_dns بالprimary dns الخاص بمحتوى الحماية اللي انت مختاره و نفس الكلام على الsecondary

  3. User info

    3. أول حاجة هتكتب الكلام ده

  4. User info

    4. netsh interface ip set dns { interface } static { primary_dns }

  5. User info

    5. خلينا نفترض مثلا انك عايز تفعل الحماية على الinterface بتاعت الwifi و عايز حماية عالية يبقى هتكتب الأمر بالشكل ده:

  6. User info

    6. netsh interface ip set dns "Wi-Fi" static 15.184.191.201
